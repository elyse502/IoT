# App Java

A small Java application skeleton with build, run, and contribution instructions. Use this README as a starting point for documentation and project onboarding.

## Prerequisites

- Java Development Kit (JDK) 11 or later
- Git (optional)

## Quick start

NetBeans:
create new project in netbeans called `AppJava`

Clone the repository (if applicable):

```bash
git clone <repo-url>
cd appjava
cp src/* <path of your new project>
```

## Project layout

```groovy
src/
    main/
        java/        Application source code
        resources/   Static resources and default configs
    test/          Unit tests
pom.xml or build.gradle
README.md
```

## Testing

Run unit tests:

NetBeans:
Open netbean > source packages > src > appjava

Right-click on file > Run file

## Contribution

- Fork the repository
- Create a feature branch
- Add tests and documentation
- Open a pull request describing your changes

## Contact

Open an issue or pull request in the repository for questions, bug reports, or feature requests.
